# Whatsapp Bot - sticker creator

## Getting Started

This project require NodeJS v12.

### Install
Clone this project

```bash
> git clone https://github.com/asinjery/botwa.git
> cd botwa

```

Install the dependencies:

```bash
> npm i
```

### Usage
1. run the Whatsapp bot

```bash
> npm start
```

after running it you need to scan the qr
